# Traffic Management System (C++)

This project is developed as part of the **1st Year BSc IT curriculum**.
It demonstrates the use of **Object-Oriented Programming** and **File Handling**
in C++ through a real-world Traffic Management System.

## Features
- Add vehicle details
- Issue traffic challans
- Store data permanently using files
- View stored vehicle and challan records
- Menu-driven console application

## Technologies Used
- C++
- File Handling (ifstream, ofstream)
- Object-Oriented Programming (OOP)
